# README
This personal website is based on [PRISM](https://prism-demo.pages.dev/). I modified some parts for improving bibtex, and adding a feature for daily google scholar stats update.

## Usage
`Use this template` -> `crete a new repository` -> '[Your user name].github.io'

`Settings` -> `Secrets and variables` -> `Actions` -> `Repository secrets` -> `get an API key from https://serpapi.com/` -> `SERPAPI_KEY`

`Settings` -> `Actions` -> `General` -> `Workflow permissions` -> `Read and write permissions` -> `âˆš Allow GitHub Actions to create and approve pull requests`
